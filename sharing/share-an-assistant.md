# Share an agent

Use the **Share** tab to invite a collaborator to an agent.

## Add a collaborator

1. Open the agent workspace.
2. Open **Share**.
3. Enter the collaborator's email address.
4. Add their name if you want. Name is optional.
5. Choose **Add collaborator**.

After the collaborator is added, they appear in the collaborator list.

Shared agents may require a plan that includes sharing. If an invite says the owner needs to upgrade, the owner should open **Settings** -> **Billing** and choose a plan that includes **Shared agents**.

## Send an invite link

Choose **Invite link** for the collaborator. Aubrey shows an invite link that you can copy and send.

## Pair Telegram or WhatsApp

Choose **Telegram** or **WhatsApp** for the collaborator if you want to create a messaging link for them.

## Remove access

Choose **Remove** when the collaborator should no longer have access to that agent.
